Conceptual Overview

During this unit, I came to understand that a curve is something I can write in more than one way, and to switch between different forms so that I can solve problems more efficiently. With parametric equations x(t)x(t)x(t) and y(t)y(t)y(t), I focus on how the coordinates change together but as their own individual relationship with a new variable.

From x′(t)x'(t)x′(t) and y′(t)y'(t)y′(t), I can read the instantaneous speed v(t)=(x′(t))2+(y′(t))2v(t)=\sqrt{(x'(t))^2+(y'(t))^2}v(t)=(x′(t))2+(y′(t))2​, which turns arc length into the equation: L=∫v(t) dtL=\int v(t)\,dtL=∫v(t)dt, when thinking about it in a velocity and distance scenario. Importance & Applications (Paraphrased from Sources)

  • Physics and Engineering: Path length and speed show up in robot/vehicle motion planning, where if you re-parameterize arc-length, it makes following a path smoother and timing much more aware (Wen et al., 2021,Ali et al., 2020).
  • Computer Graphics and Animation: Many systems use approximately arc-length-parameterized splines to move objects at visually constant speed and to space textures evenly (Wang & Kearney, 2003;NVIDIA SIGGRAPH, 2020).
  • Navigation and Radar: Polar coordinates (range, angle) are the standard way to describe targets. Air-traffic and weather radar use a polar format before any map reprojection (Research Starters;AMS Journal, 2016).
